How to permanently save changes
Saving keywords to GitHub
Changes made on this page are saved in your browser session only. To permanently add keywords so they survive page refresh:
1. Go to github.com/labs-boomsustainable/vitics
2. Click vitics-keywords.js → click the pencil (edit) icon
3. Find the right category array and add your keyword in quotes with a comma
4. Click Commit changes — Vitics picks it up automatically
